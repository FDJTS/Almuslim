Install Almuslim

Windows
  1. Download the installer from the Downloads page.
  2. Run the installer and follow the steps.
  3. If SmartScreen appears, choose “More info” → “Run anyway”.
Linux

Option A: AppImage

chmod +x ./Almuslim-*.AppImage
./Almuslim-*.AppImage

Option B: Debian/Ubuntu (.deb)

sudo apt install ./almuslim_amd64.deb
macOS
  1. Open the .dmg and drag the app to Applications.
  2. If blocked, allow it in System Settings → Privacy & Security.
Android
  1. Download the APK.
  2. Enable install from unknown sources if prompted.
  3. Open the APK to install.
